## üí° Tips for Best Results

### For DALL-E 3:
- Use the exact prompts above
- Request "photorealistic" and "high quality"
- Ask for "no text or branding"

### For Midjourney:
- Add `--ar 4:5` for standard views
- Add `--ar 2:5` for sleeve views  
- Add `--ar 4:3` for hat
- Use `--v 6` for latest model
- Add `--style raw` for product photography look
- Example: `[prompt] --ar 4:5 --v 6 --style raw --q 2`

### For Stable Diffusion:
- Use **Realistic Vision** or **DreamShaper** models
- CFG Scale: 7-9
- Steps: 30-50
- Sampler: DPM++ 2M Karras
