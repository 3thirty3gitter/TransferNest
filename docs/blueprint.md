# **App Name**: TransferNest

## Core Features:

- Image Upload: Allow users to upload images for transfer printing.
- Sheet Width Selection: Allow customers to select a sheet width of either 13 or 17 inches.
- Automated Image Nesting tool: tool to Automatically arrange uploaded images efficiently onto the selected sheet width using generative AI reasoning.
- Live Sheet Layout Display: Display a preview of the sheet layout in real time as images are uploaded and nested.
- Shopping Cart: Maintain user cart during image upload and sheet arranging.
- Checkout Process: Integrate with Square for payment processing.
- Order Submission: Allow user to generate print-ready file to submit for fulfillment

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust in the printing process.
- Background color: Light gray (#EEEEEE), provides a neutral backdrop that makes the transfers stand out and ensures a clear viewing experience.
- Accent color: Teal (#009688), highlighting key interactive elements such as the image upload button and call to actions, creating an encouraging touch for conversion
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text. Allows the UI to convey a friendly, simple feel
- Utilize a grid-based layout to ensure a clean and organized presentation of images and sheet configurations. The image nest array will fill the center of the layout, allowing a customer to instantly preview a graphical view.
- Employ clear and intuitive icons for actions such as upload, delete, and arrange, which provide additional context without relying heavily on written descriptions.
- Use subtle transitions and animations to indicate loading states and process completion (e.g., fade-in effects for newly uploaded images) to offer feedback to users, reducing any wait time tension.
